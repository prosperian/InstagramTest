# InstagramTest
In this project I used instagram graphql api to retrieve user posts and save them in Pictures directory
## Technologies and libraries 
- Android
- Kotlin
- MVVM
- Dependency Injection 
- Retrofit
- Coroutines
- Glide
- Dagger 
- ViewModel lifycycle
